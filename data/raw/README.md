### Training Dataset

The dataset used in this project is publicly available and was obtained from Kaggle:

- **Name:** AI vs Human Text  
- **Source:** https://www.kaggle.com/datasets/shanegerami/ai-vs-human-text

The dataset contains textual samples labeled according to their origin (human-written or AI-generated) and is used for training and evaluation text classification models.

### Test Dataset

The dataset used in this project is publicly available and was obtained from Kaggle:

- **Name:** LLM - Detect AI Generated Text Dataset  
- **Source:** https://www.kaggle.com/datasets/sunilthite/llm-detect-ai-generated-text-dataset

The dataset contains textual samples labeled according to their origin (human-written or AI-generated) and is used for evaluating text classification models.
